Struct Space 

Source 
pub struct Space<E: SpaceElement> { /* private fields */ }
Available on crate feature desktop only.
Expand description

Represents two dimensional plane to map windows and outputs upon.

Space is generic over the types of elements mapped onto it. The simplest usecase is a Space<Window>, but other types can be used by implementing SpaceElement. Multiple types might be quickly aggregated into an enum by using the space_elements!-macro.

pub fn element_under<P: Into<Point<f64, Logical>>>( &self, point: P, ) -> Option<(&E, Point<i32, Logical>)>

Finds the topmost element under this point if any and returns it together with the location of this element relative to this space.

This is equivalent to iterating the elements in the space from top to bottom and testing if the point is within the elements input region and returning the first matching one.

Note that SpaceElement::is_in_input_region expects the point to be relative to the elements origin.
