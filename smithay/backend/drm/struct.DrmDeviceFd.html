pub fn new(fd: DeviceFd) -> DrmDeviceFd

Create a new DrmDeviceFd.

This function will try to acquire the master lock for the underlying drm device and release the lock on drop again. For that reason you should never create multiple DrmDeviceFd out of the same DeviceFd, but instead clone the DrmDeviceFd.

Failing to do so might fail to acquire set lock and release it early, which can cause some drm ioctls to fail later.
