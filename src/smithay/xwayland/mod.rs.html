3//! XWayland utilities
4//!
5//! This module contains helpers to manage XWayland from your compositor, in order
6//! to support running X11 apps.
7//!
8//! The starting point is the [`XWayland`] struct, which represents the
9//! running XWayland instance. Dropping it will shutdown XWayland.
10//!
11//! You need to provide an implementation of a X11 Window Manager for XWayland to
12//! function properly. You'll need to treat XWayland (and all its X11 apps) as one
13//! special client, and play the role of an X11 Window Manager.
14//!
15//! Smithay does not provide any helper for doing that yet, but it is planned.
16mod x11_sockets;
17mod xserver;
18pub mod xwm;
19
20pub use self::xserver::{XWayland, XWaylandClientData, XWaylandEvent};
21pub use self::xwm::{X11Surface, X11Wm, XwmHandler};
